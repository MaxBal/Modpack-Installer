# Pterodactyl panel addon minecraft mods and modpacks installer
Pterodactyl panel addon minecraft mods and modpacks installer

This addon is made for the current 1.12.1 version of the Pterodactyl Panel

# Installation
1. cd /var/www/pterodactyl
2. git clone https://github.com/MaxBal/Modpack-Installer.git
3. cd Modpack-Installer
4. sudo sh addon.sh
